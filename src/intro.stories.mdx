## Amplify components Storybook

This is the storybook for the `@equinor/amplify-components` npm-package.

In here you will find components that are used in the amplify team developed applications in Equinor.
Everything from small utilities we use, like `formatDate()` and bigger things like the `SelectField` component.

This project is maintained and developed by and for the amplify team.

You can find the repository [here](https://github.com/equinor/amplify-components).
