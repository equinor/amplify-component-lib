## Usage

`Breadcrumbs` are often placed horizontally before a page's main content.

```tsx
import { Breadcrumbs } from '@equinor/amplify-component-lib';

<Breadcrumbs>
  <Breadcrumbs.Breadcrumb href="#">Item1</Breadcrumbs.Breadcrumb>
  <Breadcrumbs.Breadcrumb href="#">Item2</Breadcrumbs.Breadcrumb>
  <Breadcrumbs.Breadcrumb href="#" aria-current="page">
    Item3
  </Breadcrumbs.Breadcrumb>
</Breadcrumbs>;
```

### Wrapped

`Breadcrumbs` by default wraps to the next line if the parent container is too narrow.
With the `wrap` property set to false, the `Breadcrumbs` will instead automatically truncate.
It might be a good idea to use `forceTooltip` on the `Breadcrumbs.Breadcrumb` when this is the case.

### Third-party routing libraries

Use the `as` prop on `Breadcrumbs.Breadcrumb` to utilize third party routers such as react-router with `Breadcrumbs`

```tsx
import { Link } from 'react-router-dom';
import { Breadcrumbs } from '@equinor/amplify-component-lib';

<Breadcrumbs>
  <Breadcrumbs.Breadcrumb as={Link} to="/">
    Home
  </Breadcrumbs.Breadcrumb>
  <Breadcrumbs.Breadcrumb as={Link} to="/about" aria-current="page">
    About
  </Breadcrumbs.Breadcrumb>
</Breadcrumbs>;
```
